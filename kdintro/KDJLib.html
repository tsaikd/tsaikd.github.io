在寫 Java project 的過程中
覺得有些東西可以寫的比較通用化
就有了一系列的 Maven library 產生了

KDJLib_parent
下面幾個 Project 的 Maven parent

KDJLib
主要的 Maven Project
有些好用的工具

ArgParser 可以解析 common line 程式的參數
ConfigUtils 可以方便的取出設定檔的資料
ProcessEstimater 可以用來計算程式還要多久才能跑完, 處理大量資料的時候好用
XPathUtils 把解析 XML 的工具再包一層, 讓它更好用

KDJLib_Web
在寫 Java Web 的時候常用到的工具
filter 系列有產生 Etag 的工具, 壓縮網頁的工具
mongodb 系列可以拿來傳 json 的資料, 當然也能跟 MongoDB 溝通
servlet 系列有簡單的 Proxy 工具, 取網頁參數的工具
utils 系列有包裝過的 Http Client 工具

KDJLib_SolrClient
在跟 Solr 溝通時可能會用到的東西
